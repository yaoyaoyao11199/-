# 响应式电商网站

## 项目说明

这是一个基于HTML5、CSS3和JavaScript (ES6+)开发的响应式电商网站，实现了商品展示、购物车、价格筛选和搜索等核心功能。网站采用纯前端技术栈，无需后端服务器，使用localStorage实现数据持久化。

## 功能特性

### 1. 商品展示页面
- 响应式栅格布局，在桌面端显示4列商品，平板端显示2-3列，移动端显示1-2列
- 每个商品卡片包含商品图片、名称、价格、评分和"加入购物车"按钮
- 实现商品分类导航和分页功能
- 确保在所有设备尺寸下布局合理、视觉美观

### 2. 购物车功能
- 使用localStorage实现购物车状态的本地持久化存储
- 支持添加商品、修改数量、删除商品和清空购物车操作
- 实时计算商品小计、总计金额和商品总数
- 实现购物车悬浮图标及数量提示，点击展开购物车详情

### 3. 价格筛选和搜索功能
- 实现价格区间滑块筛选（最小值-最大值范围选择）
- 实现商品名称关键词搜索功能，支持模糊匹配
- 筛选和搜索结果需实时更新，无需页面刷新
- 添加搜索历史记录功能，支持一键重新搜索

### 4. 图片懒加载优化
- 实现基于Intersection Observer API的图片懒加载
- 为未加载图片提供适当的占位符或骨架屏
- 优化图片加载性能，包括适当的图片格式（WebP优先）和尺寸
- 实现图片加载失败的优雅降级处理

## 技术栈

- **基础技术：** HTML5、CSS3、JavaScript (ES6+)
- **图片服务：** 使用Trae API生成商品图片
- **存储：** localStorage本地存储
- **响应式设计：** CSS Grid和Flexbox

## 安装步骤

1. **克隆或下载项目**
   - 将项目文件下载到本地目录

2. **直接打开**
   - 在浏览器中直接打开 `index.html` 文件即可运行
   - 无需安装任何依赖或构建工具

## 使用指南

### 浏览商品
- 在首页浏览商品列表
- 使用左侧分类导航筛选不同类别商品
- 使用价格滑块筛选特定价格区间的商品
- 使用顶部搜索框搜索商品

### 购物车操作
- 点击商品卡片上的"加入购物车"按钮添加商品
- 点击右上角购物车图标查看购物车详情
- 在购物车中修改商品数量、删除商品或清空购物车
- 查看购物车总计金额

### 响应式体验
- 在不同设备尺寸下测试网站响应式布局
- 体验从移动端到桌面端的流畅过渡

## 性能优化

- **图片懒加载：** 仅当图片进入视口时才加载
- **CSS优化：** 使用CSS Grid和Flexbox实现响应式布局
- **JavaScript优化：** 避免不必要的DOM操作和重绘
- **本地存储：** 使用localStorage减少重复计算和网络请求

## 浏览器兼容性

- 支持所有现代浏览器（Chrome、Firefox、Safari、Edge）
- 对于不支持Intersection Observer API的浏览器，提供了降级方案

## 质量与性能要求

- **页面加载时间：** 首次内容绘制(FCP) < 1.5秒，最大内容绘制(LCP) < 2.5秒
- **响应式设计：** 完美适配320px-1920px宽度的设备
- **代码规范：** 遵循W3C标准，确保无语法错误
- **可访问性：** 符合WCAG 2.1 AA级标准，支持键盘导航
